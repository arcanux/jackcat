 From Pillars Made of Sand.

For the quick walkthrough... (see attached map)

When you get in, go to the east, and hit the lever I have an arrow pointing at. Then you go back to the middle and go up the ramp numbered 4 (SE ramp up). I didn't draw that room because it's kind of straight forward. But there will be a door open (or if it closed because you're slow) that will say "Can not be activated from here." On the other side of this door there is a Wretched key on the floor. Get this key then go back to the drop point and then go up the NW ramp labeled 1. In this room there is a door that the Wretched key fits, and behind that door is a Depraved key. Go back to the drop again, and go back to the SE ramp (labeled 4) and find the door that depraved key fits. Behind that door there will be an Abyssal Key, get one, then head back to the drop. Then one last time go to the NW ramp (labeled 1) and find the door that key fits. Behind that door will be a Pandemic Key. Get alot of these keys! There is no timer on them and they open the door you need to get through to get to the end and if you have alot of these keys you don't have to do everything in this paragraph over, if you die =P

Okay! now that you have your Pandemic Keys, go back to the drop. Go to either the NE or SW (I'd reccommend SW because you won't have electrical shock damage) and you will come to 4 doors that the pandemic key will fit. It doesn't matter which door you use it on, they all lead to the same place, which is a pit.

The rest is VERY straight forward and self explanatory, so I'm not going to explain it! =P There is Yaja the Shepard (drops the spear, which is on a week timer) and Ya'ruldi, who drops about 12 keys (week timer) that you can use on Yaruldi's Horde Chests in the bottom. Each key has 3 uses.

There ya go!
