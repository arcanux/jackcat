Note from Sept 5th:
I've been out of town and will return at the end of the week! I'll post the reports of the live event with Grael's forces when I return! If you have a report or screens of a live event, please send them in and I'll add them to the live events page soon!! Thanks!

August 25, 2006
Monday's weekly update has been added to the Dev Updates page. The Live Events page has also been updated with the assault on Sawato by Lei Wan-tzu!

August 21, 2006
Numerous reports added to the Live Events page, including information on a new title. To whet your appetite, an excerpt from Clav of WE's report from the Black Totem quest: "For the record, I want it noted I got the kill using a sling shot. :p".

August 20, 2006
The Live Events page has been updated with numerous reports, first from the Grael appearances yesterday, than today with the opening of the totem dungeons! Did you receive one of the globals? Send your report in!

August 17, 2006
It's patch day! Yay! Turbine will be performing additional server maintenance during the downtime, bso it has now been extended to 7am EDT - 6pm EDT. I'll be updating the Shattering the Dark patch page as I'm able this evening as I'm able. Please include full stats, up close poses and your in-game name and server with submissions! Thanks!

August 14, 2006
The weekly update is available on the Dev Updates page here. The Letter to the Players and the Event Teaser for Thursday's event, Shattering the Dark, are also available here and here!

August 11, 2006
The exclusive teaser pic for August's event, Shattering the Dark, is now available here!

August 7, 2006
This week's update has been added to the Dev Updates page! Of note is news of a token that may be redeemed for a new pack doll or a level to compensate for recent downtime!

The Rares page has also been updated with rare #238, Ibriya's Choice, a necklace discovered by N Oo B of DT!

August 4, 2006
The Live Events page has received numerous updates -- first a few more reports of Grael's first appearances, then 2 accounts of the new quest on Asheron's Island (with lots of screenshot)! The Dev Updates page has been updated with several of the Devs recent posts, including this week's update! Finally, the Rares page has been updated with rare #103, the Hieroglyph of Fealty, found by Zolatar of LC on an Olthoi Ripper.
